# 易编科技 技术博客

## 如何开始
```
// 进入目录并安装依赖
cd blog && npm install

// 启动项目
npm start
```
打开浏览器，然后通过 http://localhost:8080/blog/ 访问

## 部署
在根目录下执行：
```
npm run pd
```
访问地址：
[https://yibiankeji.github.io/blog/](https://yibiankeji.github.io/blog/)

## 修改首页
使用 编辑器 打开 项目后，通过 src/README.md 文件可以修改首页的名称和logo等。

通过 src/.vuepress目录下得config.js 文件，可以修改导航栏的内容。
